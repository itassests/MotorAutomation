🏢 Company Margin

Org-wide expected margin %. Used by the Margin Exceptions tab to flag per-rule margins that drift away from this baseline. GLOBAL is the fallback; vehicle-type rows (CAR / TW / GCV / PCV / MISC) override it for that type.

Scope Margin % Notes Updated Action

🔒 Cycle Snapshots

Every cycle that has a stored bulk-calculation snapshot. Finalized cycles reject all further row-edits, recompute, CQB / Cancel, Move and wipe. Admins can Unfinalize any cycle from here when something genuinely needs to change. Finalizing is done from the Bulk Calculation tab so the rows + totals can be reviewed before locking.
